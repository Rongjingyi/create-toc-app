# create-toc-app 
<br />

Create-toc-app is the global command for create a toc app;  
More about toc.js, see [here](https://github.com/junjie-lean/toc.js/tree/alpha).

## Installanion

Using npm:
```javascript

    npm install -g npm
    npm install -g create-tpc-app

```

## Create Project

In Terminal or CMD:
```javascript

    create-toc-app <project Name>

```
